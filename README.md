# AGÓN - Sala de Juegos

## Descripción
Aplicación web desarrollada en Angular que simula una sala de juegos con autenticación y navegación protegida.

## Tecnologías
- Angular
- TailwindCSS
- RxJS

## Funcionalidades
- Login y registro con validaciones
- Protección de rutas
- Página "Quién Soy" con integración a GitHub API
- Sistema de notificaciones (Toast)
- UI estilo gaming (inspirado en Opera GX)

## Instalación
npm install  
ng serve

## Deploy
URL: https://agon-sand.vercel.app/
